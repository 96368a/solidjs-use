# useAsyncValidator

Wrapper for [`async-validator`](https://github.com/yiminghe/async-validator).

## Install

```bash
npm i solidjs-use @solidjs-use/integrations async-validator
```

## Usage

```ts
import { useAsyncValidator } from '@solidjs-use/integrations/useAsyncValidator'
```
